# Create Counter React App
create react app to render 2 button one to increase number one to decrease number between 0 to 10 styling with tailwindcss

## Counter React App Screenshoot
![Counter React App Screenshoot](src/assets/img/CounterApp.png)

## How to run app
- npm i -D vite @vitejs/plugin-react
- npm i react react-dom tailwindcss @tailwindcss/vite
- npm run dev
- open localhost:5173 in browser
